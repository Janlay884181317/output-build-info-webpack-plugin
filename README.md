# output-build-info-webpack-plugin

### 使用方式

```shell
npm install output-build-info-webpack-plugin --save-dev
```


```javascript
const OutputBuildInfoWebpackPlugin = require('output-build-info-webpack-plugin');
```

```
module.exports = {
    // ....
    plugins: [
        new OutputBuildInfoWebpackPlugin()
    ]
};
```
